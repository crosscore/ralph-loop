## Task Description

Add a user authentication feature to the application.

## Requirements

- Users should be able to sign up with email and password
- Users should be able to log in and log out
- Session should persist across page refreshes
- Add appropriate error handling for invalid credentials

## Notes

- Use JWT for session management
- Follow existing code patterns in the project
